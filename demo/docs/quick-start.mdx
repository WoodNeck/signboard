First, you should put a canvas element inside your page.

```html
<canvas id="signboard"></canvas>
```

Then initialize Sign instance with JavaScript after.

```js
import SignBoard from "signboard";

const signboard = new SignBoard("#signboard", "PATH_TO_YOUR_IMAGE/VIDEO", {
  contentType: "image",
  frameRate: 60,
  autoResize: true,
  tileSize: 8,
  emission: 1,
  bulbSize: 0.7
});
```
